# TelnetSever

How to run server

```
sudo apt install gcc-multilib g++-multilib
https://github.com/xybyn/TelnetSever.git
cd TelnetSever
mkdir build 
cd build
cmake ..
make
./TelnetDebugger 2424
```

2424 - port of the server

How to connect to the server
`
telnet localhost 2424
`
